# Task-1-Karabo-Mahlapha
DecodeLabs Task 1 repository
This is a repository for DecodeLabs Artificial Intellence week 1 internship project. For this project, the task was to create a rule-based chatbot. The ruled-based chatbot was created in this project for a Kiosk that sells healthy foods such as salads, sandwhich, smoothie, fruitbowls and yogurt to workers in CBD. The chatbot assists customers in finding the menu and foods that are available for the lunch so that they know what can find from the kiosk before they can leave their offices so that they do not waste. The rule-based chatbot is coded in the jupyter notebook file in the repository.
